Для сборки проекта:
```
mkdir build
cd build
cmake ..
make
```

Будет создано несколько исполняемых файлов, среди которых ```apotheosis_server``` и ```apotheosis_algorithm```.
Для запуска сервера (из директории build/):
```
./apotheosis_server --config ../server/configs/config.yaml
```
